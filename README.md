# 🖥️ VDI Nexus 

A real-time VDI (Virtual Desktop Infrastructure) allocation dashboard. This tool allows the team to manage a pool of virtual machines, assign users dynamically, and monitor live statuses (Active, Idle, Offline).

---

## 🚀 Key Features
* **Live Syncing:** Updates automatically across all team members' screens every 5 seconds.
* **Dynamic Inventory:** Add or remove VDIs directly from the dashboard.
* **Smart Assignment:** One-click user assignment with automatic status transitions.
* **Privacy-First:** Hosted on a secure GitHub link with a Supabase cloud database backend.

## 🛠️ How to Use
1. **Assign a VDI:** Select your name from the dropdown next to an available machine. The status will automatically turn **Active**.
2. **Release a VDI:** Change the user back to **Available** when finished. The status will return to **Offline**.
3. **Add New VDI:** Use the "Add" panel at the top to register new hostnames into the system.
4. **Maintenance:** Use the status dropdown to flag machines undergoing updates.
